Tutorial

Getting started

This page explains the shared layout and how to add multiple tutorial pages with a consistent look.

Introduction

Each tutorial page uses the same header, the same stylesheet, and the same TOC behavior. You write content in normal HTML (headings, paragraphs, lists, code blocks).

Tip Keep section ids short and stable (they become your TOC links).

Folder structure

tutorial/
  index.html
  getting-started.html
  building-your-first-deck.html
  troubleshooting.html
  assets/
    tutorial.css
    tutorial.js

You can add more pages next to the existing ones in tutorial/.

Add a new page

  1. Copy a page. Duplicate building-your-first-deck.html to a new file name.

  2. Update the TOC. Edit the left TOC links and section ids so they match.

  3. Add a hub card. Add a new <a> card on tutorial/index.html.

Prev/next navigation

At the bottom of each tutorial page, there’s a “pager” with links to the previous and next tutorials so readers can follow a linear path.
