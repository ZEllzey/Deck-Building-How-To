**What is this?**

This is a guide for Magic: The Gathering player who wants to learn to build their own deck in some of the most popular 
formats of Magic: The Gathering. It includes how-to guides for building a commander deck, standard decks, and how to 
build a deck at a pre-release event. It goes through the basics of building each deck type and tips for card selection.

**Why does this exist?**

I created this as I have found there are plenty of tools out there to use to build Magic decks, but there are not really 
any guides on the strategy or approach to how you build decks. So I decided to create a tool that players can use to learn 
how to approach building their Magic decks to be able to create decks that are enjoyable to play and appropriately competitive
for the level that their group is playing.

**What tools did I use?**

I used Cursor as the code editor for this project. I also used the AI assistant within Cursor to help create the main template 
and functions. I was working in HTML and using CSS sheets that were all deployed through GitHub. I decided to use HTML and CSS 
sheets as I was familiar with them and felt comfortable using them. I decided to use Cursor and the embedded AI assistant, as it 
would be able to assist me in fixing bugs, as well as help create the code for more complex features I wanted to add, but did not 
feel confident I would be able to create on my own in the allotted timeframe.
