**Decisions:**

The most important decisions I made while doing this project were mainly related to what content would and would not be included. There is a lot of detailed information that could be
included in this website about deck building strategies, and the specifics on each deck format that was included. Though I decided to keep the information fairly limited as I intended
it to be a guide, not an instruction manual. I also decided to gear this towards an audience that does have some experience with the topic, as this is not a task that new players would
likely take on, and the needed explanation for a lot of Magic: The Gathering lingo would likely become overwhelming for that audience. Trying to accommodate the website for new players
would also require a lot more content to be added, which would create difficulties in being able to finish this project in the time allotted.  

**What Worked?:**

Using the AI to make the template for the webpage worked very well, as it was able to understand the basics of what I needed for the webpage. I also was able to include instructions to 
allow me to easily edit and add to the webpage. I am very satisfied with the visuals and interactive elements I was able to create with the AI assistance that I could adapt to my purposes. 
I also feel that the way I organized the information made it very intuitive and easy to understand. I am very satisfied with the overall product I have created, as I feel it would be 
actually useful to the audience it is intended for.

**What would you do differently?:**

If I were to do this again, I would probably write up the content separately from the Code editor, as it makes it difficult to edit the text for grammar and spelling. I continued to do 
this as it helps my brain keep track of where I am, but it proved difficult to type content efficiently. I think I would use the same tools, but if I started over, I would start more 
specific towards that AI, so there are no unneeded features or functions that I have to remove. The AI did a good job of creating a template, but if I redid it, I would have a better 
understanding of how the AI assistant works, so I could use it to start closer to my final goal.

**What did you learn?:**

I learned a lot about how to format text in a webpage to make it easy to view and read across multiple viewing platforms. I also learned that I tend to try to make the first draft the
only draft when I complete things, so I had to practice heavy discipline to make sure I didn’t spend too long on one aspect that could be fixed later on. I learned more about how to use
HTML and the capabilities of AI in coding HTML. I was able to learn the tendencies and more about how the AI approaches issues, and learned how to ask for exactly what I wanted. I also 
learned that AI is not good at creating text, as it kept trying to autocomplete my sentences and would be very far off in its suggestions. I also learned it is very important to decide 
the scope of the project from the beginning, as it can lead to wasting time on an aspect that ends up being out of the final scope. I ultimately learned that spending time on a topic and
problem that I am very passionate about and interested in makes it much easier to create a product I actually care about.
